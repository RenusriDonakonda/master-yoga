# 🧘 Master Yoga

Master Yoga is a modern web application designed to help users explore yoga practices, discover poses, improve wellness, and maintain a healthy lifestyle through an intuitive and responsive user experience.

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js (v18 or later recommended)
- npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/RenusriDonakonda/master-yoga.git
```

Move into the project directory:

```bash
cd master-yoga
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

or

```bash
npm start
```

Open:

```text
http://localhost:3000
```
